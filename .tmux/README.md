# tmux

## 101

- See [Ham Vocke's guide](https://www.hamvocke.com/blog/a-quick-and-easy-guide-to-tmux/)

> Split pane

```
C-b %
```

> Close pane

```
C-b x
```

> cli: `<prefix> : <cmd>`
```
C-b : <cmd>
```

> List all sessions

```
tmux ls
```

> Detach session
```
C-b d
```

// To detach session and choose, `C-b D`.

> Kill all sessions

```
tmux kill-server
```
